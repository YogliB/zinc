# Suggested Commands

- `cargo new --lib crates/agent-core` : Create agent-core crate.
- `cargo build` : Build the project.
- `cargo run --bin mcp-server` : Run MCP server.
- `npm create tauri-app apps/zinc-ide` : Create Tauri app (once Node setup).
- `npm run tauri dev` : Run IDE in dev mode.
- `git add . && git commit -m "message"` : Commit changes (no --no-verify).
