# Codebase Structure

- `crates/agent-core/`: Shared agent core with tools.
- `crates/mcp-server/`: MCP server binary.
- `crates/ide/`: Tauri + SvelteKit app with global shortcuts (Cmd+O/Ctrl+O for folder, Cmd+Shift+O/Ctrl+Shift+O for file) and get_os command.
- `docs/`: Documentation, masterplans, plans.
- Root: AGENTS.md, README.md, settings.json eventually.
